# Документация проекта

## Навигация

- `repository_structure.md` — карта репозитория (где что лежит).
- `tasks_overview.md` — как читать выполненные задания.
- `tz_compliance_report.md` — соответствие ТЗ.
- `user_guide_testing.md` — инструкция для проверки продукта.
- `deploy_public.md` — деплой в публичный доступ для заказчика.
- `specs/README.md` — все исходные материалы заказчика в одном месте.

## Техническая часть

- `architecture.md` — архитектура.
- `domain_model.md` — доменные сущности.
- `rbac.md` — роли и доступы.
- `business_rules.md` — бизнес-ограничения.
- `navigation_map.md` — карта экранов/меню.
- `assumptions.md` — допущения и оговорки.
