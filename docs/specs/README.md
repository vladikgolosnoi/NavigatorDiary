# Исходные материалы ТЗ

В этой папке собраны все исходные файлы заказчика в одном месте.

## Состав

- `01_technical_spec_initial.pdf` — исходная версия технического задания.
- `02_technical_spec_updated.pdf` — обновленная версия технического задания.
- `03_appendix_2_goals_catalog.pdf` — Приложение 2 (каталог целей).
- `04_appendix_3_specialties.docx` — Приложение 3 (специальности и чек-листы).
- `05_appendix_badges.pdf` — файл со значками и визуальными элементами.

## Как использовать

- Для сверки реализации с требованиями: сначала открывать `02_technical_spec_updated.pdf`, затем приложения.
- Для обновления контента проекта:
  - цели — из `03_appendix_2_goals_catalog.pdf`;
  - специальности — из `04_appendix_3_specialties.docx`;
  - значки/графика — из `05_appendix_badges.pdf`.
