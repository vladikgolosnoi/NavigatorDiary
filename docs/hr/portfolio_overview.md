# Портфолио-обзор проекта

## Что это за проект

`Дневник навигатора` — мобильное PWA-приложение для подростков и молодежи с ролевой системой:

- навигатор;
- руководитель команды;
- организатор.

Проект реализует полный цикл: регистрация, каталог целей и специальностей, прогресс, достижения, чат команды, дополнительные сервисы.

## Что здесь можно оценить как HR/техлиду

- Работа с fullstack-архитектурой (React + NestJS + PostgreSQL + Prisma).
- Проработка RBAC и ролевых сценариев без «заглушек».
- Реализация бизнес-ограничений:
  - до 12 целей за выбор;
  - повторный выбор через 3 месяца;
  - прогресс цели не чаще 1 раза в неделю;
  - одна активная специальность.
- Мобильный UX и PWA-подход.
- Чистая проектная документация и трассировка требований до реализации.
