## Как быстро снять аккуратные скриншоты

1. Запустить проект локально (`README.md` → «Локальный запуск»).
2. Открыть страницу в браузере в мобильном режиме (ширина 390px).
3. Снять скриншот без лишних панелей/вкладок.
4. Сохранить файл строго под указанным именем.

Рекомендуется единый размер: `1170x2532` или `1242x2688`.
